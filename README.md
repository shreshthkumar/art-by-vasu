# Art by Vasu — Mandala Canvas Art Website

A hand-crafted e-commerce website for original mandala canvas paintings.

## Pages

- **Home** (`index.html`) — Hero, featured paintings, about snippet, testimonials
- **Shop** (`shop.html`) — Full gallery with filters and cart
- **About** (`about.html`) — Artist story, process, facts
- **Contact** (`contact.html`) — Contact form and FAQ accordion

## Customisation

- Edit `data/paintings.json` to update your painting inventory
- Replace placeholder gradients with real photos in `images/`
- Update contact email in `contact.html`
- Update personal facts in `about.html`

## Stack

Plain HTML · CSS · Vanilla JS — no build step required.
